# 62. Unique Paths

## 解題程式碼

```javascript
var uniquePaths = function (m, n) {
  const DP = new Array(m).fill().map(() => new Array(n).fill(1));

  for (let i = 1; i < DP.length; i++) {
    for (let j = 1; j < DP[i].length; j++) {
      DP[i][j] = DP[i - 1][j] + DP[i][j - 1];
    }
  }

  return DP[m - 1][n - 1];
};
```
