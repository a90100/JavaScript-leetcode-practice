# 1957. Delete Characters to Make Fancy String

## 解題程式碼

```javascript
var makeFancyString = function (s) {
  let l = 0;
  let res = '';

  for (let i = 0; i < s.length; i++) {
    if (s[i] !== s[l]) l = i;
    if (i - l >= 2) continue;
    res += s[i];
  }

  return res;
};
```
