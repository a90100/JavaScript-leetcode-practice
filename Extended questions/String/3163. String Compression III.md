# 3163. String Compression III

## 解題程式碼

```javascript
var compressedString = function (s) {
  let res = '';
  let l = 0;

  for (let i = 0; i < s.length; i++) {
    if (s[i] !== s[l] || i - l === 9) {
      res += `${i - l}${s[l]}`;
      l = i;
    }
  }

  return (res += `${s.length - l}${s[s.length - 1]}`);
};
```
