# 374. Guess Number Higher or Lower

## 解題程式碼

```javascript
var guessNumber = function (n) {
  let l = 0;
  let r = n;

  while (l <= r) {
    const mid = Math.floor((l + r) / 2);
    const res = guess(mid);
    if (res === 0) return mid;
    if (res === 1) l = mid + 1;
    if (res === -1) r = mid - 1;
  }
};
```

## 解題思路、演算法

就是做 binary search。

## 解法的時間、空間複雜度

時間複雜度: O(log n)
空間複雜度: O(1)
