# 2690. Infinite Method Object

## 解題程式碼

```javascript
function createInfiniteObject() {
  return new Proxy(
    {},
    {
      get: function (target, propKey) {
        return function () {
          return String(propKey);
        };
      },
    }
  );
}
```
