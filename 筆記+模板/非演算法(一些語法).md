## 取亂數

> 生成 0 ~ x 間的亂數

```javascript
function getRandom(x) {
  return Math.floor(Math.random() * x);
}
```

## 取出將 x 開根號後的最大整數


## 最小公倍數、最大公因數

```javascript
function gcd(a, b) {
  while (b !== 0) {
    let temp = b;
    b = a % b;
    a = temp;
  }
  return a;
}

function lcm(a, b) {
  return Math.abs(a * b) / gcd(a, b);
}
```
