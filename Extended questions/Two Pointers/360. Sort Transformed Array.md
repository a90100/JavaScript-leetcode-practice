# 360. Sort Transformed Array

## 題目說明

Given a **sorted** integer array `nums` and three integers `a`, `b` and `c`, apply a quadratic function of the form `f(x) = ax^2 + bx + c` to each element `nums[i]` in the array, and return _the array in a sorted order_.

Example 1:

```
Input: nums = [-4,-2,2,4], a = 1, b = 3, c = 5
Output: [3,9,15,33]
```

Example 2:

```
Input: nums = [-4,-2,2,4], a = -1, b = 3, c = 5
Output: [-23,-5,1,7]
```

Constraints:

```
1 <= nums.length <= 200
-100 <= nums[i], a, b, c <= 100
nums is sorted in ascending order.
```

**Follow up**: Could you solve it in `O(n)` time?

## 解題思路、演算法

推導：

```
ex 1：
(-1) * (-4)^2 + 3 * (-4) + 5 = -23
(-1) * (-3)^2 + 3 * (-3) + 5 = -13
(-1) * (-2)^2 + 3 * (-2) + 5 = -5
(-1) * (-1)^2 + 3 * (-1) + 5 = 1
(-1) * (0)^2 + 3 * (0) + 5 = 5
(-1) * (1)^2 + 3 * (1) + 5 = 7
(-1) * (2)^2 + 3 * (2) + 5 = 7
(-1) * (3)^2 + 3 * (3) + 5 = 5
(-1) * (4)^2 + 3 * (4) + 5 = 1
(-1) * (5)^2 + 3 * (5) + 5 = -5
(-1) * (6)^2 + 3 * (6) + 5 = -13
(-1) * (7)^2 + 3 * (7) + 5 = -23
好像有對稱

ex 2：
1 * (-5)^2 + 3 * (-5) + 5 = 15
1 * (-4)^2 + 3 * (-4) + 5 = 9
1 * (-3)^2 + 3 * (-3) + 5 = 5
1 * (-2)^2 + 3 * (-2) + 5 = 3
1 * (-1)^2 + 3 * (-1) + 5 = 3
1 * (0)^2 + 3 * (0) + 5 = 5
1 * (1)^2 + 3 * (1) + 5 = 9
1 * (2)^2 + 3 * (2) + 5 = 15
1 * (3)^2 + 3 * (3) + 5 = 23

確定有對稱，只是可能是 大 => 小 => 大，或相反
```

這樣可以先判斷 nums 陣列第一個元素和中間元素經過公式計算後的值，看是 `大 => 小 => 大` 或 `小 => 大 => 小`，然後設定指針去指向最小的計算值的索引。
