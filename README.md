## 每份筆記可能會記錄以下幾項

1. 解題的程式碼，也包括在其他平台上，高手公開發表的程式碼(附上來源)
2. 解題思路(解題步驟，用什麼演算法解決，極限值/特殊狀況)、參考的解題資料
3. 該解法的時間、空間複雜度
4. 有需要再多加練習的題目記得標記
5. 其他類(額外補充之類的資訊)
