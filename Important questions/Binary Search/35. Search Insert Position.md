# 35. Search Insert Position

## 解題程式碼

```javascript
var searchInsert = function (nums, target) {
  let l = 0;
  let r = nums.length - 1;

  while (l <= r) {
    const mid = Math.floor((l + r) / 2);
    if (nums[mid] === target) return mid;
    if (nums[mid] > target) r = mid - 1;
    if (nums[mid] < target) l = mid + 1;
  }
  return l;
};
```

## 解法的時間、空間複雜度

時間複雜度: O(log n)
空間複雜度: O(1)
