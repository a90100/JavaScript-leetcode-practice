# 1047. Remove All Adjacent Duplicates In String

## 解題程式碼

```javascript
var removeDuplicates = function (s) {
  const stack = [];

  for (let i = 0; i < s.length; i++) {
    if (stack.length > 0 && stack[stack.length - 1] === s[i]) {
      stack.pop();
    } else {
      stack.push(s[i]);
    }
  }

  return stack.join('');
};
```
