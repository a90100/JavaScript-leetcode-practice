# 2914. Minimum Number of Changes to Make Binary String Beautiful

## 解題程式碼

```javascript
var minChanges = function (s) {
  let res = 0;

  for (let i = 0; i < s.length; i += 2) {
    if (s[i] !== s[i + 1]) res++;
  }

  return res;
};
```

## 解法的時間、空間複雜度

時間複雜度: O(n)
空間複雜度: O(1)
