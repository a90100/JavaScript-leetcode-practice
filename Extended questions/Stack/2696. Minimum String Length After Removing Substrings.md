# 2696. Minimum String Length After Removing Substrings

## 解題程式碼

```javascript
var minLength = function (s) {
  const stack = [s[0]];

  for (let i = 1; i < s.length; i++) {
    if ((stack[stack.length - 1] === 'A' && s[i] === 'B') || (stack[stack.length - 1] === 'C' && s[i] === 'D')) {
      stack.pop();
      continue;
    }
    stack.push(s[i]);
  }
  return stack.length;
};
```
