# 796. Rotate String

## 解題程式碼

第一次解:

```javascript
var rotateString = function (s, goal) {
  let c = 0;

  while (c < s.length) {
    let char = s.shift();
    s.push(char);
    if (s.join('') === goal.join('')) return true;
    c++;
  }
  return false;
};
```

優化:

```javascript
var rotateString = function (s, goal) {
  return s.length === goal.length && (s + s).includes(goal);
};
```
