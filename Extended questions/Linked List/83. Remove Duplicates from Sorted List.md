# 83. Remove Duplicates from Sorted List

## 解題程式碼

```javascript
var deleteDuplicates = function (head) {
  let cur = head;

  while (cur && cur.next) {
    if (cur.val === cur.next.val) {
      cur.next = cur.next.next;
    } else {
      cur = cur.next;
    }
  }

  return head;
};
```

## 解法的時間、空間複雜度

時間複雜度: O(n)
空間複雜度: O(1)
