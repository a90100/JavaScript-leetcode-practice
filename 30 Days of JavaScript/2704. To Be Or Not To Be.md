# 2704. To Be Or Not To Be

## 解題程式碼

```javascript
var expect = function (val) {
  return {
    toBe: (checkVal) => {
      if (val === checkVal) {
        return true;
      } else {
        throw new Error('Not Equal');
      }
    },
    notToBe: (checkVal) => {
      if (val !== checkVal) {
        return true;
      } else {
        throw new Error('Equal');
      }
    },
  };
};
```

## 解題思路、演算法

可以 Google function chaining

## 解法的時間、空間複雜度

時間複雜度: O(1)
空間複雜度: O(1)
