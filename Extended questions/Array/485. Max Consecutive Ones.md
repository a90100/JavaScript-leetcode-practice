# 485. Max Consecutive Ones

## 解題程式碼

```javascript
var findMaxConsecutiveOnes = function (nums) {
  let l = 0;
  let max = 0;

  for (let i = 0; i < nums.length; i++) {
    if (nums[i] === 0) {
      l = i + 1;
    } else {
      max = Math.max(max, i - l + 1);
    }
  }
  return max;
};
```

## 解法的時間、空間複雜度

時間複雜度: O(n)
